problem: >-
  Reinstalling Windows by hand meant hours of clicking through OOBE, debloating,

  changed between installs.
approach: >-
  A PowerShell image builder that takes a declarative profile and produces a
  customized Windows 11 ISO — apps, policies, and provisioning baked in offline,
  so the result is reproducible from a clean source image.
outcome: >-
  A fresh machine goes from blank disk to fully provisioned desktop in one
  unattended pass, and every install is described by a profile that lives in
  version control.
